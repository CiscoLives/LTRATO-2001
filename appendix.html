Examples¶

Configuration Compliance¶

Steps:

  1. Install jsonschema python library

    pip install jsonschema

  2. Connect to pyATS shell

    pyats shell --testbed-file pyats_testbed.yaml

  3. Import jsonschema python library

    from jsonschema import validate

  4. Define a schema for the configuration compliance test.

    schema = {
    "type": "object",
    "properties": {"description": {"type": "string"}},
    "required": ["description"],
}

  5. Define, connect, and parse the show interfaces command.

    csr = testbed.devices['csr1000v-1']
csr.connect()
csr_interfaces = csr.parse("show interfaces")

  6. Validate the parsed output against the schema.

    validate(instance=csr_interfaces["Loopback1"], schema=schema)

  7. If the validation fails, an exception will be raised.
