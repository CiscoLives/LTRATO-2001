Step 1: Access CXTM platform and create a new projectÂ¶

Value Proposition: CXTM is a cloud-based test automation platform that enables businesses to streamline their software testing processes, improve quality assurance, and accelerate time-to-market. By providing a centralized environment for managing and executing automated tests, CXTM helps organizations reduce the costs associated with manual testing, increase testing coverage, and enhance the overall efficiency of their development cycles.

  1. Open a Chrome browser and access the CXTM platform using the following credentials:

    • Username cisco

    • Password cisco

  2. To start utilizing CXTM, you must create a project. Projects are comprised of test cases and automation job files, that are executed to obtain results. Click Sandbox tab on the main page or Projects in the left-hand navigational panel, and then click on NEW PROJECT button.

  3. Fill in the required information to instantiate the project. Then, click the Submit button. These details are customizable per project and/or customer.

    • Project Name: LTRATO-2001

  4. You will now see your project in the menu and are ready to start creating test cases and the current view is your project dashboard.
